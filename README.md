# Meus Mapas

Sistema full stack para criação de mapas e cadastro de pontos fisicoa.

## Tecnologias
- Back-end: PHP 8 + Laravel
- Front-end: React + Tailwind CSS
- Banco de dados: (definir)
- Mapas: OpenStreetMap + Leaflet

## Descrição
O sistema permite criar mapas lógicos e cadastrar pontos físicos associados, exibindo-os em um mapa interativo.
